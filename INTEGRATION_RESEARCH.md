# 외부 플랫폼 구독 연동 조사

> 작성일: 2026-04-02
> 목적: 카카오/구글/앱스토어/플레이스토어 등 외부 플랫폼의 구독 정보를 SubFlow로 가져올 수 있는지 조사

---

## 1. 연동 가능성 요약

| 플랫폼 | 가능 여부 | 비고 |
|--------|-----------|------|
| Gmail 영수증 파싱 | ✅ 가능 | OAuth + Gmail API, 결제 이메일 파싱 |
| Google Play 구독 (모바일) | ✅ 가능 | BillingClient로 해당 기기 활성 구독 조회 |
| App Store 구독 (모바일) | ✅ 가능 | StoreKit 2로 현재 기기 구독 조회 |
| 카카오톡 계정 구독 | ❌ 불가 | 사용자 구독 목록 조회 API 없음 |
| 카카오페이 정기결제 | ⚠️ 제한적 | 가맹점만 조회 가능, 사용자 전체 내역 불가 |
| 앱스토어 전체 구독 | ❌ 불가 | Apple이 제3자에게 사용자 구독 목록 제공 안 함 |
| 플레이스토어 전체 구독 | ❌ 불가 | Google도 제3자 앱에 전체 구독 노출 안 함 |
| 카드사 결제 내역 | ❌ 불가 (인가 필요) | 마이데이터 사업자 등록 필요 (금융위 인가) |

---

## 2. 상세 분석

### 2.1 Gmail 영수증 파싱 (웹 + 모바일)

- **방법**: Google OAuth → Gmail API로 결제 영수증 이메일 검색 → 파싱하여 구독 정보 추출
- **얻을 수 있는 정보**: 서비스명, 결제 금액, 결제일, 결제 주기
- **장점**: 웹에서도 가능, 대부분의 구독 서비스가 이메일 영수증 발송
- **단점**: 이메일 파싱 정확도 이슈, 사용자가 이메일 권한 동의 필요
- **기술 스택**: Google OAuth 2.0, Gmail API, 이메일 파싱 로직

### 2.2 Google Play 구독 조회 (모바일 전용)

- **방법**: `BillingClient.queryPurchasesAsync(BillingClient.ProductType.SUBS)`
- **얻을 수 있는 정보**: 활성 구독 목록, 구독 상태, 갱신일
- **장점**: 공식 API, 정확한 데이터
- **단점**: 모바일 앱에서만 가능, 해당 기기의 구독만 조회
- **기술 스택**: `react-native-iap` 또는 Expo In-App Purchases

### 2.3 App Store 구독 조회 (모바일 전용)

- **방법**: StoreKit 2의 `Transaction.currentEntitlements`
- **얻을 수 있는 정보**: 현재 활성 구독, 갱신일, 가격
- **장점**: 공식 API, 정확한 데이터
- **단점**: iOS에서만 가능, 해당 Apple ID의 구독만
- **기술 스택**: `react-native-iap` 또는 Expo In-App Purchases

### 2.4 카카오페이 정기결제

- **방법**: 카카오페이 정기결제 API
- **한계**: 가맹점(서비스 제공자)만 자기 서비스의 정기결제를 조회할 수 있음. 사용자의 전체 정기결제 내역을 제3자가 조회하는 API는 제공되지 않음.

### 2.5 카드사 결제 내역 (마이데이터)

- **방법**: 금융위원회 마이데이터 사업자 인가 → 오픈뱅킹 API
- **얻을 수 있는 정보**: 카드 결제 내역에서 정기결제 패턴 자동 감지
- **한계**: 사업자 인가 절차 필요 (뱅크샐러드, 토스 등이 이 방식)
- **현실성**: 개인/소규모 팀에서는 사실상 불가

## 3. 구현 우선순위 제안

### 1단계 — 즉시 가능
- **수동 입력 + 서비스 DB 매칭** (현재 방식 유지)
- **Gmail 영수증 파싱**: Google OAuth 연동으로 결제 이메일 자동 감지

### 2단계 — 모바일 앱 확장
- **Google Play / App Store 구독 조회**: `react-native-iap` 활용
- Expo 프로젝트가 이미 있으므로 모바일에서 기기 구독 정보 연동 가능

### 3단계 — 장기 (사업 확장 시)
- 마이데이터 사업자 인가 → 카드 결제 내역 기반 구독 자동 감지
